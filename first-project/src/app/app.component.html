Services

Angular Notations:

  • @NgModule: Angular Module
  • @Component: Angular Component
  • @Injectable: Angular Service

@NgModule consist of different parts:

  • declarations: Contain all the components used by the module.
  • imports: Contain all other modules needed for the module.
  • providers: Contain all the services used by the module.

Create a dependency injection:

  1. Create a service: ng generate service service_name
  2. Services that are used in a module are declared inside: 'providers' array in app.module
  3. Import the service in the component where the service is used in app.component
  4. Inject the service in the constructor as:
    constructor(svc: TestService)

Result: {{service}}

HttpClient

Steps:

  1. Import the module: HttpClientModule from '@angular/common/http'
  2. Import the angular module in imports array under @NgModule
  3. Import in the component: HttpClient from '@angular/common/http'
  4. Inject in the contructor to instantiate: private http: HttpClient
  5. Make a http call like get request: this.http.get('https://api.github.com/users/gurvsxen487')
    - It makes a asynchronous call.
    - The asynchronous call return a observable (Promises in angular 1)
    - We can excute a function to make sure the asynchronous call is done.
    - It is the done by subscribe function present in http service.
  6. We can see the response by the response object by passing it as a argument in subscribe function.
  7. See the result in the console.
