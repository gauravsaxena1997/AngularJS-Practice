Services

Angular Notations:

  • @NgModule: Angular Module
  • @Component: Angular Component
  • @Injectable: Angular Service

@NgModule consist of different parts:

  • declarations: Contain all the components used by the module.
  • imports: Contain all other modules needed for the module.
  • providers: Contain all the services used by the module.

Create a dependency injection:

  1. Create a service: ng generate service service_name
  2. Services that are used in a module are declared inside: 'providers' array in app.module
  3. Import the service in the component where the service is used in app.component
  4. Inject the service in the constructor as:
    constructor(svc: TestService)

Result: {{service}}
