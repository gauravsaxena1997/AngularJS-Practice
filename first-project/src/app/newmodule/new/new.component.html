Component of another Module

  1. Create a module: "ng generate module module_name"
  2. Create a component in the new module : "ng generate component module_name/component_name"
  3. Export the component from module_name.module.ts: exports: [ NewComponent],
  4. Import the new module in app.module.ts:
    - imports: [NewmoduleModule]
